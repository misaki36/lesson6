# 自己紹介ページ

自分についての情報を紹介するWebページです。

## 📝 プロジェクト説明

このプロジェクトは、プログラミング学習の過程での最初のWebページ制作課題です。

基本的なHTML・CSSを使用して、以下の情報をまとめたポートフォリオサイトとなっています：

- 名前と自己紹介
- 趣味の一覧
- スキルレベルの表示（プログレスバー付き）
- 外部リンク

## 🛠 使用技術

- **HTML** - ページ構造の作成
- **CSS** - スタイリングとアニメーション効果

### 実装の特徴

- レスポンシブデザイン対応
- グラデーション背景
- CSSアニメーション（フェードイン、プログレスバー展開）
- ホバーエフェクト
- Google Fontsの活用（Noto Sans JP）
